# Seedance 2.0 Fast 首尾帧模式使用指南

基于官方手册 + 社区实战经验整理，供 Writer Agent 生成 videoPrompt 时参考。

## 核心原则

1. **首尾帧定画面，Prompt 定动作** — 画面内容已在图片中，videoPrompt 只描述变化
2. **@1 = 首帧，@2 = 尾帧** — 必须用 @ 语法引用素材
3. **保持一致** — 必须包含"保持构图和色彩一致"
4. **一个镜头指令** — 不要多个冲突的运镜
5. **慢优于快** — slow/smooth/gentle，避免 fast

## Prompt 公式（首尾帧模式）

```
@1 @2 [动态变化描述]，保持构图和色彩一致，镜头[运镜]，[风格]，avoid jitter and bent limbs
```

### 公式拆解

| 元素 | 要求 | 示例 |
|------|------|------|
| 素材引用 | @1 @2 必须在最前 | @1 @2 |
| 动态变化 | 描述从首帧到尾帧的变化，用具体动词 | 人物缓缓转头，微风吹过头发，从左走到右 |
| 保持一致 | 必须包含 | 保持构图和色彩一致 |
| 运镜 | 只用一个主指令 | 镜头缓慢推进 / 镜头固定不动 / 镜头缓慢右移 |
| 风格 | 简短 | 电影级质感 / 柔和治愈系 / 赛博朋克风 |
| 约束 | 避免常见问题 | avoid jitter and bent limbs |

### 长度控制

- 最佳：60-100 词（中文约 40-70 字）
- 太短：缺少关键信息
- 太长：指令冲突，质量下降

## 8 种运镜

| 运镜 | 中文 | 适用场景 |
|------|------|---------|
| 推镜头 | push-in / 缓慢推进 | 情感聚焦、特写 |
| 拉镜头 | pull-out / 缓慢拉远 | 环境展现 |
| 横移 | pan / 横向平移 | 跟随主体 |
| 跟踪 | tracking / 跟随镜头 | 动作场景 |
| 环绕 | orbit / 环绕 | 产品展示、人物肖像 |
| 航拍 | aerial / 俯拍 | 风景、城市 |
| 手持 | handheld / 手持感 | 纪实风格 |
| 固定 | fixed / 固定不动 | 专注主体动作 |

**规则**：只用一个主运镜！多个冲突运镜 = 抖动/混乱

## 速度关键词

| 速度 | 关键词 | 风险 |
|------|--------|------|
| 极慢 | imperceptible, barely | 安全 |
| 慢 | slow, gentle, gradual | ✅ 推荐 |
| 中 | smooth, controlled | 安全 |
| 快 | dynamic, swift | ⚠️ 高风险，只用一个元素 |

**铁律**：fast 是最容易翻车的关键词。如果需要快节奏，只让一个元素快。

## 光线（杠杆最高的元素）

加一句光线描述 > 加十个形容词。

| 光线 | 效果 | 示例 |
|------|------|------|
| golden hour | 温暖金色 | 柔和的黄金时段光线 |
| rim light | 边缘高光 | 戏剧性边缘光 |
| natural light | 自然光 | 柔和自然窗光 |
| neon | 霓虹 | 霓虹灯雨天街道 |
| backlit | 逆光 | 逆光剪影 |
| overcast | 阴天漫射 | 柔和漫射光 |

## 负向提示词（必加）

| 负向提示 | 排除 |
|----------|------|
| avoid jitter | 画面抖动 |
| avoid bent limbs | 肢体扭曲 |
| avoid temporal flicker | 时间闪烁 |
| avoid identity drift | 角色特征漂移 |

## Prompt 模板

### 模板 1：人物微动作
```
@1 @2 画面中的人物缓缓转头看向镜头，微风吹过头发，
保持构图和色彩一致，镜头缓慢推进，电影级质感，
柔和自然光，avoid jitter and bent limbs
```

### 模板 2：人物移动
```
@1 @2 人物从画面左侧缓缓走到右侧，步伐轻盈自然，
保持构图和色彩一致，镜头固定不动，治愈系风格，
温暖午后光线，avoid jitter and bent limbs
```

### 模板 3：环境变化
```
@1 @2 画面中的光影缓缓变化，从晨光变为午后暖阳，
树叶轻轻摇曳，保持构图和色彩一致，镜头缓慢拉远，
自然纪录片风格，avoid jitter and bent limbs
```

### 模板 4：情感递进
```
@1 @2 人物表情从沉思缓缓变为微笑，眼神温柔，
保持构图和色彩一致，镜头缓慢推进到特写，
电影级质感，柔和侧面光线，avoid jitter and bent limbs
```

### 模板 5：动作场景（低动作）
```
@1 @2 人物缓缓伸出手，轻轻触碰面前的物品，
保持构图和色彩一致，镜头缓慢右移跟随，
电影级色彩分级，avoid jitter and bent limbs
```

## 首尾帧图片要求

### 即梦文生图（首帧/尾帧）Prompt 要点

1. **构图**：明确主体位置（居中/偏左/远景）
2. **色彩**：描述色调（暖色/冷色/对比强烈）
3. **风格**：漫画风、写实、电影感
4. **细节**：角色服装、表情、发型、场景元素
5. **光线**：描述光源方向和质感
6. **连贯性**：首帧和尾帧同一场景同一角色，状态有变化

### 首帧 vs 尾帧的区别

| 维度 | 首帧 | 尾帧 |
|------|------|------|
| 人物位置 | 起始位置 | 结束位置（移动后） |
| 表情 | 起始表情 | 结束表情（变化后） |
| 光线 | 起始光线 | 结束光线（可变化） |
| 构图 | 基本构图 | 保持一致但可有微小变化 |
| 场景 | 同一场景 | 同一场景 |

## 大神技巧

### 分段生成
3 段 5 秒 > 1 段 15 秒。前段截图作后段参考。

### 情感递进
按秒数描述：0-2s 温柔 → 2-5s 升温 → 5-10s 高潮 → 10-15s 回归

### 素材优先级
人物参考图 > 运镜参考视频 > 配乐 > 其他细节
